Querying XBRL reports submitted to the SEC

This tutorial is up-to-date against the 3.4 Irvine release, published on May 20th, 2014.

This repository contains:

  • A tutorial in PDF format for making oneself familiar with the XBRL connector on top of the 28.io platform. A direct link is here

  • The queries used in this tutorial. These queries can be directly uploaded to a 28.io project using the 28 command line. How to connect to an XBRL/NoLAP repository is explained in the PDF above.
